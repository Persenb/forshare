大学物理速查表

一、刚体力学

1. 刚体转动的能量

  • 转动动能： \(E_k = \dfrac{1}{2}J\omega^2\)
  • 动能定理： \(\int M d\theta = \dfrac{1}{2}J\omega_2^2 - \dfrac{1}{2}J\omega_1^2\)
  • 滚动（纯滚动）： \(E_k = \dfrac{1}{2}mv_c^2 + \dfrac{1}{2}J\omega^2,\quad v_c = \omega R\)

2. 角动量定理

  • 质点角动量： \(\vec{L} = \vec{r} \times m\vec{v}\)
  • 刚体定轴角动量： \(L = J\omega\)
  • 角动量定理： \(M = \dfrac{dL}{dt} = J\alpha\)
  • 角动量守恒： \(M_{\text{外}} = 0 \Rightarrow J_1\omega_1 = J_2\omega_2\)

3. 转动定律 \(M = J\alpha\)

  • \(M = J\alpha\)（类比 \(F = ma\)）
  • 解题：平动列 \(F=ma\)，转动列 \(M=J\alpha\)，约束 \(a = R\alpha\)

二、静电场

4. \(E\) 的计算式

  • 定义： \(\vec{E} = \vec{F}/q_0\)
  • 点电荷： \(E = \dfrac{q}{4\pi\varepsilon_0 r^2}\)
  • 叠加原理： \(\vec{E} = \sum\vec{E}_i\) 或 \(\vec{E} = \int d\vec{E}\)
  • 常见分布：
    • 无限长均匀带电直线：\(E = \dfrac{\lambda}{2\pi\varepsilon_0 r}\)
    • 无限大均匀带电平面：\(E = \dfrac{\sigma}{2\varepsilon_0}\)
    • 均匀带电圆环轴线：\(E = \dfrac{qx}{4\pi\varepsilon_0(R^2+x^2)^{3/2}}\)

5. 电偶极子

  • 电偶极矩： \(\vec{p} = q\vec{l}\)（\(-q \to +q\)）
  • 均匀外场中力矩： \(\vec{M} = \vec{p} \times \vec{E}\)，\(M = pE\sin\theta\)
  • 电势能： \(W = -\vec{p}\cdot\vec{E} = -pE\cos\theta\)
  • 轴线上电场： \(E = \dfrac{1}{4\pi\varepsilon_0}\dfrac{2p}{r^3}\)
  • 中垂面上电场： \(E = \dfrac{1}{4\pi\varepsilon_0}\dfrac{p}{r^3}\)

6. 电场的高斯定理

  • 真空： \(\oint \vec{E}\cdot d\vec{S} = \dfrac{q_{\text{内}}}{\varepsilon_0}\)
  • 介质： \(\oint \vec{D}\cdot d\vec{S} = q_{\text{内}}\)，\(\vec{D} = \varepsilon_0\varepsilon_r\vec{E}\)
  • 选择高斯面：球对称→同心球面、轴对称→同轴圆柱面、面对称→柱形高斯面

7. 电场的环路定理

  • \(\oint \vec{E}\cdot d\vec{l} = 0\) → 静电场是保守场
  • 电场力做功与路径无关，可引入电势

8. 电势的计算

  • 定义： \(U_P = \displaystyle\int_P^{\infty} \vec{E}\cdot d\vec{l}\)
  • 点电荷： \(U = \dfrac{q}{4\pi\varepsilon_0 r}\)
  • 叠加原理（标量）： \(U = \sum U_i\) 或 \(U = \int dU\)
  • 均匀带电球壳： 内部 \(U = \dfrac{q}{4\pi\varepsilon_0 R}\)（等势），外部 \(U = \dfrac{q}{4\pi\varepsilon_0 r}\)
  • 电场与电势关系： \(\vec{E} = -\nabla U\)，\(E_x = -\partial U/\partial x\)

9. 静电平衡

  • 条件： 导体内部 \(E = 0\)，导体是等势体
  • 电荷分布： 只在外表面，曲率越大 \(\sigma\) 越大
  • 表面附近： \(E = \dfrac{\sigma}{\varepsilon_0}\)，方向垂直表面
  • 空腔导体（静电屏蔽）： 内表面无电荷，腔内无电场

10. 静电能与电场能量密度

  • 电容器储能： \(W = \dfrac{1}{2}QU = \dfrac{1}{2}CU^2 = \dfrac{Q^2}{2C}\)
  • 电场能量密度： \(w_e = \dfrac{1}{2}\varepsilon_0\varepsilon_r E^2 = \dfrac{1}{2}\vec{E}\cdot\vec{D}\)
  • 总静电能： \(W_e = \displaystyle\int_V \dfrac{1}{2}\varepsilon_0 E^2 dV\)

三、静磁场

11. 毕奥-萨伐尔定律

  • 微分： \(d\vec{B} = \dfrac{\mu_0}{4\pi}\dfrac{Id\vec{l}\times\vec{r}}{r^3}\)
  • 积分： \(\vec{B} = \dfrac{\mu_0}{4\pi}\displaystyle\int\dfrac{Id\vec{l}\times\vec{r}}{r^3}\)
  • 无限长直导线： \(B = \dfrac{\mu_0 I}{2\pi a}\)
  • 圆环圆心： \(B = \dfrac{\mu_0 I}{2R}\)
  • 圆弧圆心： \(B = \dfrac{\mu_0 I}{2R}\cdot\dfrac{\theta}{2\pi}\)

12. 密绕螺线管磁场

  • 内部： \(B = \mu_0 n I\)（\(n = N/L\)）
  • 端点： \(B = \dfrac{1}{2}\mu_0 n I\)
  • 外部： \(\approx 0\)

13. 安培环路定理

  • \(\oint \vec{B}\cdot d\vec{l} = \mu_0 \sum I_{\text{内}}\)
  • 方向：右手四指沿绕行方向，拇指指向与电流同向则 \(I > 0\)

14. 无限长载流圆柱导体磁场

  • 内部（\(r < R\)）： \(B = \dfrac{\mu_0 I r}{2\pi R^2}\)
  • 表面（\(r = R\)）： \(B = \dfrac{\mu_0 I}{2\pi R}\)
  • 外部（\(r > R\)）： \(B = \dfrac{\mu_0 I}{2\pi r}\)

四、磁场对电流的作用

15. 安培力、磁力矩、磁力的功

  • 安培力： \(d\vec{F} = I d\vec{l} \times \vec{B}\)
  • 磁矩： \(\vec{p}_m = NIS\vec{n}\)
  • 磁力矩： \(\vec{M} = \vec{p}_m \times \vec{B}\)，\(M = NISB\sin\theta\)
  • 磁力的功： \(A = I\Delta\Phi\)

五、电磁感应

16. 法拉第定律与楞次定律

  • 法拉第定律： \(\mathcal{E}_i = -\dfrac{d\Phi}{dt}\)（多匝：\(\mathcal{E}_i = -N\dfrac{d\Phi}{dt}\)）
  • 楞次定律： 感应电流的效果反抗引起感应电流的原因（增反减同）

17. 动生电动势与感生电动势

  • 动生： \(\mathcal{E} = \displaystyle\int (\vec{v}\times\vec{B})\cdot d\vec{l}\)，直导线：\(\mathcal{E} = Blv\)
  • 感生： \(\mathcal{E}_i = -\dfrac{d\Phi}{dt} = -\displaystyle\int_S \dfrac{\partial\vec{B}}{\partial t}\cdot d\vec{S}\)
  • 涡旋电场： \(\oint \vec{E}_{\text{涡}}\cdot d\vec{l} = -\displaystyle\int_S \dfrac{\partial\vec{B}}{\partial t}\cdot d\vec{S}\)

18. 麦克斯韦方程组

  • \(\oint\vec{D}\cdot d\vec{S} = q_0\) — 电场有源
  • \(\oint\vec{B}\cdot d\vec{S} = 0\) — 磁场无源
  • \(\oint\vec{E}\cdot d\vec{l} = -\int\dfrac{\partial\vec{B}}{\partial t}\cdot d\vec{S}\) — 变化磁场产生电场
  • \(\oint\vec{H}\cdot d\vec{l} = I_0 + \int\dfrac{\partial\vec{D}}{\partial t}\cdot d\vec{S}\) — 电流和变化电场产生磁场
  • 位移电流： \(I_d = \varepsilon_0\dfrac{d\Phi_e}{dt}\)

六、波动光学

19. 杨氏双缝干涉

  • 明纹： \(d\sin\theta = \pm k\lambda\)（\(k=0,1,2,\ldots\)）
  • 暗纹： \(d\sin\theta = \pm (2k-1)\dfrac{\lambda}{2}\)（\(k=1,2,\ldots\)）
  • 条纹间距： \(\Delta x = \dfrac{D}{d}\lambda\)

20. 光程差与相位差

  • 光程： \(nr\)
  • 相位差： \(\Delta\varphi = \dfrac{2\pi}{\lambda}\delta\)
  • 干涉判据： \(\delta = \pm k\lambda\)（明），\(\delta = \pm(2k+1)\dfrac{\lambda}{2}\)（暗）

21. 薄膜干涉

  • 反射光光程差： \(\delta = 2ne + \dfrac{\lambda}{2}\)（或 \(0\)，取决于半波损失）
  • 半波损失： 光从光疏→光密反射时，相位突变 \(\pi\)，附加 \(\lambda/2\)

22. 迈克尔逊干涉仪

  • \(\Delta d = N\cdot\dfrac{\lambda}{2}\)（\(N\) 为条纹移动数）
  • 用于测量微小位移或波长

23. 劈尖干涉

  • 相邻条纹对应厚度差： \(\Delta e = \dfrac{\lambda}{2}\)
  • 相邻条纹间距： \(l = \dfrac{\lambda}{2\theta}\)
  • 条纹向棱边弯→凹，背离棱边弯→凸
  • 测细丝直径： \(d = \dfrac{\lambda}{2}\cdot\dfrac{L}{l}\)

24. 单缝衍射与光栅衍射

  • 单缝暗纹： \(a\sin\theta = \pm k\lambda\)（\(k=1,2,\ldots\)）
  • 光栅方程： \(d\sin\theta = \pm k\lambda\)（\(k=0,1,2,\ldots\)）
  • 缺级： \(\dfrac{d}{a} = \dfrac{k}{k'}\)（光栅级次/单缝级次为整数比）
  • 最高级次： \(k_{\max} = \lfloor d/\lambda\rfloor\)

25. 马吕斯定律

  • \(I = I_0\cos^2\alpha\)
  • 自然光通过偏振片：\(I = I_0/2\)

26. 布儒斯特定律

  • \(\tan i_B = \dfrac{n_2}{n_1}\)，\(i_B + r = 90^\circ\)
  • 反射光为完全偏振光（\(\perp\) 入射面），折射光为部分偏振光

七、气体动理论

27. 理想气体状态方程

  • \(pV = \dfrac{m}{M}RT = \nu RT\)
  • \(p = nkT\)（\(n\) 分子数密度）
  • 等温 \(p_1V_1 = p_2V_2\)，等压 \(V_1/T_1 = V_2/T_2\)，等容 \(p_1/T_1 = p_2/T_2\)

28. 压强公式与温度公式

  • 压强： \(p = \dfrac{1}{3}nm\overline{v^2} = \dfrac{2}{3}n\overline{\varepsilon_k}\)
  • 温度： \(\overline{\varepsilon_k} = \dfrac{3}{2}kT\)

29. 能量均分定理

  • 每个自由度 \(\dfrac{1}{2}kT\)
  • 分子平均动能： \(\overline{\varepsilon} = \dfrac{i}{2}kT\)
  • 理想气体内能： \(E = \nu\cdot\dfrac{i}{2}RT\)
  • 分子自由度： 单原子 \(i=3\)，双原子刚性 \(i=5\)，多原子刚性 \(i=6\)

31. 速率分布

  • 归一化： \(\displaystyle\int_0^\infty f(v)dv = 1\)
  • 最概然速率： \(v_p = \sqrt{\dfrac{2RT}{M}}\)
  • 平均速率： \(\bar{v} = \sqrt{\dfrac{8RT}{\pi M}}\)
  • 方均根速率： \(\sqrt{\overline{v^2}} = \sqrt{\dfrac{3RT}{M}}\)
  • 大小关系： \(v_p < \bar{v} < \sqrt{\overline{v^2}}\)

八、热力学

32. 热力学过程

  • 第一定律： \(Q = \Delta E + A\)，\(\Delta E = \nu C_V\Delta T\)（任何过程）
  • 摩尔热容： \(C_V = \dfrac{i}{2}R\)，\(C_p = C_V + R\)，\(\gamma = \dfrac{C_p}{C_V} = \dfrac{i+2}{i}\)
过程 \(\Delta E\) \(A\) \(Q\)
等容 \(\nu C_V\Delta T\) \(0\) \(\nu C_V\Delta T\)
等压 \(\nu C_V\Delta T\) \(p\Delta V\) \(\nu C_p\Delta T\)
等温 \(0\) \(\nu RT\ln\dfrac{V_2}{V_1}\) \(A\)
绝热 \(\nu C_V\Delta T\) \(-\Delta E\) \(0\)
  • 泊松公式： \(pV^\gamma = const\)，\(V^{\gamma-1}T = const\)

33. 循环效率

  • 热机效率： \(\eta = \dfrac{A}{Q_1} = 1 - \dfrac{Q_2}{Q_1}\)
  • 制冷系数： \(e = \dfrac{Q_2}{A} = \dfrac{Q_2}{Q_1 - Q_2}\)

34. 卡诺循环

  • 卡诺热机： \(\eta = 1 - \dfrac{T_2}{T_1}\)
  • 卡诺制冷机： \(e = \dfrac{T_2}{T_1 - T_2}\)
  • 卡诺定理： 可逆机效率最高，\(\eta_{\text{不可逆}} < \eta_{\text{可逆}} = \eta_{\text{卡诺}}\)

九、振动与波

35. 简谐振动

  • 动力学： \(F = -kx\)，\(\dfrac{d^2x}{dt^2} + \omega^2 x = 0\)
  • 运动学： \(x = A\cos(\omega t + \varphi)\)，\(\omega = \sqrt{k/m}\)
  • 速度： \(v = -\omega A\sin(\omega t + \varphi)\)
  • 加速度： \(a = -\omega^2 A\cos(\omega t + \varphi)\)
  • 能量： \(E_k = \dfrac{1}{2}m\omega^2 A^2\sin^2(\omega t + \varphi)\)，\(E_p = \dfrac{1}{2}kA^2\cos^2(\omega t + \varphi)\)，\(E = \dfrac{1}{2}kA^2\)（守恒）

36. 旋转矢量法

  • 矢量长 \(A\)，角速度 \(\omega\)，与 \(x\) 轴夹角 \(\omega t + \varphi\)
  • 用于求初相、相位差、时间间隔

37. 简谐振动的合成

  • 同方向同频率： \(A = \sqrt{A_1^2 + A_2^2 + 2A_1A_2\cos\Delta\varphi}\)
  • \(\Delta\varphi = 0\)：\(A = A_1 + A_2\)（加强）；\(\Delta\varphi = \pi\)：\(A = |A_1 - A_2|\)（减弱）
  • 拍频： \(v = |\nu_1 - \nu_2|\)

38. 波函数

  • \(y(x,t) = A\cos(\omega t - kx + \varphi)\)，\(k = \dfrac{2\pi}{\lambda}\)
  • \(u = \dfrac{\omega}{k} = \lambda\nu\)
  • “−” → 波沿 \(+x\) 方向传播，“+” → 沿 \(-x\) 方向

39. 波的干涉与驻波

  • 相干条件： 同频、同方向、相位差恒定
  • 加强： \(\Delta\varphi = \pm 2k\pi\) 或 \(\delta = \pm k\lambda\)
  • 减弱： \(\Delta\varphi = \pm (2k+1)\pi\) 或 \(\delta = \pm (2k+1)\dfrac{\lambda}{2}\)
  • 驻波方程： \(y = 2A\cos\left(\dfrac{2\pi x}{\lambda}\right)\cos(\omega t)\)
  • 波腹： \(x = \pm k\cdot\dfrac{\lambda}{2}\)；波节： \(x = \pm(2k+1)\dfrac{\lambda}{4}\)
  • 相邻波腹/波节间距： \(\dfrac{\lambda}{2}\)
