大学物理复习大纲（38条全展开）

一、刚体力学

1. 刚体转动的能量

转动动能公式： \[ E_k = \frac{1}{2} J \omega^2 \]

  • \(J\) — 转动惯量，相当于”旋转中的质量”
  • \(\omega\) — 角速度，相当于”旋转中的速度”

刚体定轴转动的动能定理： \[ \int M \, d\theta = \frac{1}{2}J\omega_2^2 - \frac{1}{2}J\omega_1^2 \]

合外力矩对刚体做的功等于刚体转动动能的增量。

滚动问题中的能量守恒（纯滚动条件）： \[ E_k = \frac{1}{2}mv_c^2 + \frac{1}{2}J\omega^2,\quad v_c = \omega R \] - 第一项：质心平动动能 - 第二项：绕质心的转动动能 - 纯滚动时两者通过 \(v_c = \omega R\) 关联

⚠️ 注意区分：平动动能只与质心速度有关，转动动能只与角速度有关。滚动时两者同时存在。

2. 角动量定理

质点的角动量（对某参考点）： \[ \vec{L} = \vec{r} \times \vec{p} = \vec{r} \times m\vec{v} \] 大小 \(L = mvr\sin\theta\)，方向由右手螺旋定则判定。

刚体定轴转动的角动量： \[ L = J\omega \]

角动量定理（对轴）： \[ \vec{M} = \frac{d\vec{L}}{dt} \quad\Rightarrow\quad M = \frac{d(J\omega)}{dt} = J\alpha \]

角动量守恒定律： 合外力矩 \(\vec{M}=0\) 时，角动量守恒： \[ J_1\omega_1 = J_2\omega_2 = \text{常量} \]

常见应用场景：人坐在旋转椅上收拢/伸展双臂（改变 \(J\) 从而改变 \(\omega\)）、天体运动的掠面速度守恒、陀螺进动（了解）。

⚠️ 易错点：角动量守恒的条件是合外力矩为零，不是合外力为零。力作用线通过转轴时该力不产生力矩。

3. 转动定律 \(M = J\alpha\)

核心公式： \[ M = J\alpha \]

  • \(M\) — 合外力矩
  • \(J\) — 转动惯量（对固定转轴）
  • \(\alpha\) — 角加速度

与牛顿第二定律的类比：

平动 转动
\(F = ma\) \(M = J\alpha\)
\(m\)（质量，惯性度量） \(J\)（转动惯量，转动惯性度量）
\(v\)（速度） \(\omega\)（角速度）
\(p = mv\)（动量） \(L = J\omega\)（角动量）

解题步骤： 1. 隔离物体，受力分析 2. 对平动物体列 \(F = ma\) 3. 对转动物体列 \(M = J\alpha\) 4. 利用约束条件（如绳不可伸长、纯滚动 \(a = R\alpha\)）联立求解

⚠️ 注意：\(M = J\alpha\) 中的各量必须对 同一固定转轴 计算。绳绕过定滑轮的问题中，滑轮两侧张力一般不等（否则滑轮不会转动）。

二、静电场

4. 电场强度 \(E\) 的计算式

定义式： \[ \vec{E} = \frac{\vec{F}}{q_0} \] 单位：N/C 或 V/m

点电荷的电场： \[ E = \frac{1}{4\pi\varepsilon_0}\frac{q}{r^2} \quad（方向由 q 的正负决定） \]

场强叠加原理（核心方法）： \[ \vec{E} = \sum_i \vec{E}_i \quad\text{或}\quad \vec{E} = \int d\vec{E} \]

常见带电体的电场分布：

带电体 公式 方向
无限长均匀带电直线（线密度 \(\lambda\)） \(E = \dfrac{\lambda}{2\pi\varepsilon_0 r}\) 垂直于直线向外
无限大均匀带电平面（面密度 \(\sigma\)） \(E = \dfrac{\sigma}{2\varepsilon_0}\) 垂直于平面向外
均匀带电圆环轴线上（半径 \(R\)，总 \(q\)） \(E = \dfrac{qx}{4\pi\varepsilon_0(R^2+x^2)^{3/2}}\) 沿轴线

⚠️ 易错点：叠加时电场是矢量，要分解成分量分别计算后再合成，不要直接代数相加。

5. 电偶极子

定义： 一对等量异号点电荷 \(+q\) 和 \(-q\)，相距 \(l\)。

电偶极矩（矢量）： \[ \vec{p} = q\vec{l} \] 方向由 \(-q\) 指向 \(+q\)

电偶极子在均匀外电场中： - 受力矩： \(\vec{M} = \vec{p} \times \vec{E}\)，大小 \(M = pE\sin\theta\)，使偶极子转向外电场方向 - 电势能： \(W = -\vec{p} \cdot \vec{E} = -pE\cos\theta\) - \(\theta = 0^\circ\)（平行）：能量最低，稳定平衡 - \(\theta = 180^\circ\)（反平行）：能量最高，不稳定平衡

电偶极子轴线上的电场： \[ E = \frac{1}{4\pi\varepsilon_0}\frac{2p}{r^3} \quad (r \gg l) \]

电偶极子中垂面上的电场： \[ E = \frac{1}{4\pi\varepsilon_0}\frac{p}{r^3} \quad (r \gg l) \]

注意：电偶极子的电场与 \(1/r^3\) 成正比，衰减比点电荷（\(1/r^2\)）更快。

6. 电场的高斯定理

真空中的高斯定理： \[ \Phi_e = \oint_S \vec{E} \cdot d\vec{S} = \frac{q_{\text{内}}}{\varepsilon_0} \]

介质中的高斯定理： \[ \oint_S \vec{D} \cdot d\vec{S} = q_{\text{内}} \]

物理意义： 通过闭合曲面的电通量等于面内电荷的代数和除以 \(\varepsilon_0\)。与面外电荷无关。

选择高斯面的技巧： 1. 球对称（点电荷、均匀带电球体/球壳）→ 选同心球面 2. 轴对称（无限长直线、圆柱体）→ 选同轴圆柱面 3. 面对称（无限大平面）→ 选柱形高斯面（底面平行于平面）

⚠️ 易错点： - \(q_{\text{内}}\) 是高斯面内电荷的代数和，面外电荷不影响通量但会影响场强分布 - 高斯定理对任何闭合曲面都成立，但只有高度对称的情况才能用高斯定理求 E - \(\vec{E}\) 是曲面上各点的总场强（面内外电荷共同产生）

7. 电场的环路定理

公式： \[ \oint_L \vec{E} \cdot d\vec{l} = 0 \]

物理含义： 静电场中电场强度沿任意闭合路径的线积分为零。

推论： - 静电场是保守力场 - 静电场力做功与路径无关，只与起点和终点位置有关 - 可以引入电势（标量）来描述电场

与重力场类比：重力场中沿闭合路径重力做功为零，可以引入重力势能。

对比磁场： 磁场的环路积分 \(\oint \vec{B} \cdot d\vec{l} = \mu_0 I\) 不为零 → 磁场是非保守场（涡旋场）。

8. 电势的计算

定义： \[ U_P = \int_P^{\text{零势点}} \vec{E} \cdot d\vec{l} \] 通常取无穷远处为零势点。

点电荷的电势： \[ U = \frac{1}{4\pi\varepsilon_0}\frac{q}{r} \]

电势叠加原理（标量叠加！）： \[ U = \sum_i U_i \quad\text{或}\quad U = \int dU \]

常见带电体的电势：

带电体 电势分布
均匀带电球壳（半径 \(R\)） 内部 \(U = \dfrac{q}{4\pi\varepsilon_0 R}\)（等势），外部 \(U = \dfrac{q}{4\pi\varepsilon_0 r}\)
无限长带电直线 不能用无穷远为零势点，取某参考点 \(U = \dfrac{\lambda}{2\pi\varepsilon_0}\ln\dfrac{r_0}{r}\)

从电势求电场： \[ E_x = -\frac{\partial U}{\partial x},\quad E_y = -\frac{\partial U}{\partial y},\quad E_z = -\frac{\partial U}{\partial z} \] 或 \(\vec{E} = -\nabla U\)

⚠️ 易错点： 电势是标量，叠加时只需代数相加（比电场叠加简单得多）。电势的正负取决于场源电荷的正负和参考点的选择。

9. 静电平衡时外部电场的理解

静电平衡条件： 导体内部电场处处为零。

为什么外部电场会”变形”？

当导体放入外电场 \(E_0\) 中： 1. 导体内部的自由电子在 \(E_0\) 作用下定向移动 2. 电子移动到导体表面，产生感应电荷 3. 感应电荷在导体内部产生附加电场 \(E'\)，方向与 \(E_0\) 相反 4. 达到平衡时：\(E_0 + E' = 0\)（内部合场强为零）

静电平衡的性质： - 导体是等势体，表面是等势面 - 导体表面附近场强与表面垂直：\(E = \dfrac{\sigma}{\varepsilon_0}\) - 电荷只分布在导体表面，且曲率越大处电荷面密度越大（尖端放电的原因） - 空腔导体（法拉第笼）：内表面无电荷，腔内无电场 → 静电屏蔽

这就是为什么可以用接地的金属网罩屏蔽外部电场——导体壳内部不受外部电场影响。

10. 静电能与电场能量密度

点电荷系的静电能（互能）： \[ W = \frac{1}{2}\sum_i q_i U_i \]

电容器的储能： \[ W = \frac{1}{2}QU = \frac{1}{2}CU^2 = \frac{Q^2}{2C} \]

电场能量密度（核心公式）： \[ w_e = \frac{1}{2}\varepsilon_0 \varepsilon_r E^2 = \frac{1}{2}\vec{E}\cdot\vec{D} \]

  • 真空中：\(w_e = \frac{1}{2}\varepsilon_0 E^2\)
  • 介质中：\(w_e = \frac{1}{2}\varepsilon_0\varepsilon_r E^2\)

静电场的总能量： \[ W_e = \int_V w_e \, dV = \int_V \frac{1}{2}\varepsilon_0 E^2 \, dV \]

观点转变： 静电能不是”储存在电荷上”而是储存在电场中——这是场的观点。能量定域在电场存在的空间。

⚠️ 注意：电容器储能和电场能量密度是两种等效的计算方法，前者从电容参数出发，后者从场分布出发。

三、静磁场

11. 毕奥-萨伐尔定律

微分形式（电流元产生的磁场）： \[ d\vec{B} = \frac{\mu_0}{4\pi} \frac{I d\vec{l} \times \vec{r}}{r^3} \]

大小：\(dB = \dfrac{\mu_0}{4\pi} \dfrac{I dl \sin\theta}{r^2}\)，方向由右手螺旋定则（\(I d\vec{l}\) 叉乘 \(\vec{r}\)）确定。

积分形式： \[ \vec{B} = \int d\vec{B} = \frac{\mu_0}{4\pi} \int \frac{I d\vec{l} \times \vec{r}}{r^3} \]

常见载流导体的磁场：

(a) 直线电流 \[ B = \frac{\mu_0 I}{4\pi a}(\sin\theta_2 - \sin\theta_1) \] - 无限长直线：\(\theta_1 = -\frac{\pi}{2},\ \theta_2 = \frac{\pi}{2}\) → \(B = \dfrac{\mu_0 I}{2\pi a}\) - 半无限长：\(B = \dfrac{\mu_0 I}{4\pi a}\)

(b) 圆环电流（轴线上） \[ B = \frac{\mu_0 I R^2}{2(R^2 + x^2)^{3/2}} \] - 圆心处（\(x=0\)）：\(B = \dfrac{\mu_0 I}{2R}\) - 圆弧（圆心角 \(\theta\)）：\(B = \dfrac{\mu_0 I}{2R} \cdot \dfrac{\theta}{2\pi}\)

⚠️ 易错点： 毕萨定律涉及叉乘，方向判断是关键。直线电流用右手握住导线、拇指指向电流方向，四指绕向即磁场方向。

12. 密绕螺线管的磁场

无限长密绕螺线管内部的磁场（均匀）： \[ B = \mu_0 n I \] - \(n = N/L\) — 单位长度匝数 - \(I\) — 电流强度

螺线管外部：磁场近似为 \(0\)（理想情况）。

有限长螺线管轴线上： \[ B = \frac{\mu_0 n I}{2}(\cos\beta_1 - \cos\beta_2) \] - 端点处：\(B = \frac{1}{2}\mu_0 n I\)（是内部的一半）

右手定则： 右手四指沿电流方向弯曲，拇指指向即为螺线管内部磁场方向（N极方向）。

考题常考： 长直螺线管与环形螺线管的磁场分布对比。

13. 安培环路定理

公式： \[ \oint_L \vec{B} \cdot d\vec{l} = \mu_0 \sum I_{\text{内}} \]

物理意义： 磁感应强度沿任意闭合路径的线积分等于 \(\mu_0\) 乘以穿过该路径所围曲面的电流的代数和。

方向正负判断（右手定则）： - 右手四指沿环路积分的绕行方向弯曲 - 拇指指向与电流方向相同时 \(I > 0\)，相反时 \(I < 0\)

与高斯定理类比的应用： 与高斯定理类似，安培环路定理在高度对称的情况下可用于求 \(B\)：

对称性 选择环路 实例
轴对称 与电流同轴的圆 无限长载流直导线、圆柱体
面对称 矩形环路 无限大载流平面
螺线管对称 矩形环路 螺线管内部

⚠️ 易错点： - \(\sum I_{\text{内}}\) 是穿过环路所围曲面的电流代数和，不在环路内的电流不贡献 - \(\vec{B}\) 是环路上所有电流（包括环路外的）产生的总场，\(\mu_0\sum I_{\text{内}}\) 只等于左边的积分值

14. 无限长载流圆柱导体的磁场分布

总电流 \(I\) 均匀分布在半径为 \(R\) 的圆柱截面上。

(a) 导体内部（\(r < R\)）： \[ B = \frac{\mu_0 I r}{2\pi R^2} \] - 内部磁场与 \(r\) 成正比（从中心到表面线性增大）

(b) 导体表面（\(r = R\)）： \[ B = \frac{\mu_0 I}{2\pi R} \]

(c) 导体外部（\(r > R\)）： \[ B = \frac{\mu_0 I}{2\pi r} \] - 外部磁场与无限长直导线相同，与 \(r\) 成反比

这是典型的安培环路定理应用题。记住：内部 \(B \propto r\)，外部 \(B \propto 1/r\)。画 \(B\!-\!r\) 图是常见的考题形式。

四、磁场对电流的作用

15. 安培力、磁力矩、磁场对线圈的作用、磁力的功

安培力公式（载流导线在磁场中受力）： \[ d\vec{F} = I d\vec{l} \times \vec{B} \] 大小 \(dF = I dl B\sin\theta\)，方向由左手定则判断。

均匀磁场中的载流线圈：

(a) 安培力合力为零（线圈整体不受平移力）

(b) 磁力矩（核心公式）： \[ \vec{M} = \vec{p}_m \times \vec{B} \] \[ M = p_m B \sin\theta = NISB\sin\theta \]

  • \(\vec{p}_m = N I S \vec{n}\) — 线圈的磁矩
  • \(N\) — 匝数，\(S\) — 线圈面积，\(\vec{n}\) — 法向（右手定则确定）
  • \(\theta\) — \(\vec{n}\) 与 \(\vec{B}\) 的夹角

力矩的特征： - \(\theta = 0^\circ\)（\(\vec{n}\) 与 \(\vec{B}\) 同向）：\(M=0\)，稳定平衡 - \(\theta = 180^\circ\)（反向）：\(M=0\)，不稳定平衡 - \(\theta = 90^\circ\)：力矩最大，\(M_{\max} = NISB\)

磁力的功： \[ A = I \Delta\Phi \] - 载流导线或线圈在磁场中运动时，安培力做的功等于电流乘以磁通量的增量 - 实际上是电源克服反电动势做功的结果

左手定则： 伸左手，磁感线穿过掌心，四指指向电流方向，拇指指向安培力方向。

对比： 静电场中电偶极子的力矩 \(\vec{M} = \vec{p} \times \vec{E}\) 与磁场中线圈的力矩 \(\vec{M} = \vec{p}_m \times \vec{B}\) 形式完全对应。

五、电磁感应

16. 电磁感应与楞次定律

法拉第电磁感应定律（定义式）： \[ \mathcal{E}_i = -\frac{d\Phi}{dt} \] - \(\Phi = \int \vec{B} \cdot d\vec{S}\) — 穿过回路的磁通量 - 负号体现楞次定律

多匝线圈： \[ \mathcal{E}_i = -N\frac{d\Phi}{dt} = -\frac{d\Psi}{dt} \] - \(\Psi = N\Phi\) — 磁通链

楞次定律（方向判断）： > 感应电流的效果总是反抗引起感应电流的原因。

通俗理解： - 如果磁通量增加，感应电流产生的磁场阻碍它增加 - 如果磁通量减少，感应电流产生的磁场补偿它减少 - 即：“增反减同”

判断步骤： 1. 确定原磁场方向 2. 判断磁通量的变化（增加还是减少） 3. 根据”增反减同”确定感应电流的磁场方向 4. 用右手定则判断感应电流方向

⚠️ 易错点： 是磁通量的变化率而不是磁通量本身决定感应电动势大小。

17. 动生电动势与感生电动势

动生电动势（磁场不变，导体切割磁感线运动）： \[ \mathcal{E} = \int (\vec{v} \times \vec{B}) \cdot d\vec{l} \]

对于直导线在均匀磁场中垂直切割： \[ \mathcal{E} = Blv \] 方向由右手定则判断（\(\vec{v} \times \vec{B}\) 的方向）。

感生电动势（导体不动，磁场变化）： \[ \mathcal{E}_i = -\frac{d\Phi}{dt} = -\int_S \frac{\partial \vec{B}}{\partial t} \cdot d\vec{S} \]

涡旋电场（感生电场的本质）： 变化磁场产生的电场 \(E_{\text{涡}}\) 满足： \[ \oint \vec{E}_{\text{涡}} \cdot d\vec{l} = -\int_S \frac{\partial \vec{B}}{\partial t} \cdot d\vec{S} \]

动生电动势 感生电动势
产生原因 导体切割磁感线 磁场随时间变化
非静电力 洛伦兹力 涡旋电场力
公式 \(\mathcal{E} = Blv\) \(\mathcal{E} = -d\Phi/dt\)
适用 导体运动 导体静止

⚠️ 注意： 有时两者同时存在（如导体在变化磁场中运动），需分别计算后叠加。

18. 麦克斯韦方程组的含义

积分形式（四大方程）：

方程 名称 含义
\(\oint \vec{D} \cdot d\vec{S} = q_0\) 电场高斯定理 电场是有源场（电荷是电场的源）
\(\oint \vec{B} \cdot d\vec{S} = 0\) 磁场高斯定理 磁场是无源场（无磁单极子）
\(\oint \vec{E} \cdot d\vec{l} = -\int \frac{\partial \vec{B}}{\partial t} \cdot d\vec{S}\) 法拉第定律 变化磁场产生电场
\(\oint \vec{H} \cdot d\vec{l} = I_0 + \int \frac{\partial \vec{D}}{\partial t} \cdot d\vec{S}\) 安培环路定理（推广） 电流和变化电场都能产生磁场

核心物理思想： 1. 电场可由电荷激发（静电场），也可由变化磁场激发（涡旋电场） 2. 磁场可由电流激发（静磁场），也可由变化电场激发（位移电流） 3. 变化的电场和变化磁场相互激发 → 电磁波

位移电流： \[ I_d = \varepsilon_0 \frac{d\Phi_e}{dt} \] - 位移电流的本质是变化的电场 - 位移电流与传导电流一样产生磁场 - 位移电流不产生焦耳热

考试常考： 描述每个方程的物理意义，以及麦克斯韦方程组如何预言电磁波的存在。

六、波动光学

19. 杨氏双缝干涉

装置： 单色光通过两个相距 \(d\) 的狭缝，在后方屏幕上形成干涉条纹。

明暗纹条件： - 明纹（干涉加强）：\(\delta = d\sin\theta = \pm k\lambda,\ k = 0,1,2,\ldots\) - 暗纹（干涉减弱）：\(\delta = d\sin\theta = \pm (2k-1)\frac{\lambda}{2},\ k = 1,2,\ldots\)

条纹间距（相邻明纹或相邻暗纹的距离）： \[ \Delta x = \frac{D}{d}\lambda \] - \(D\) — 缝到屏幕的距离 - \(d\) — 双缝间距

条纹特征： - 等间距、明暗相间的直条纹 - 用白光照射时，中央为零级白色明纹，两侧彩色条纹（紫光在内，红光在外）

⚠️ 易错点： 光程差公式中的 \(k\) 从 0 开始还是从 1 开始？明纹 \(k=0\) 对应中央零级，暗纹 \(k=1\) 对应第一级暗纹。注意题目中”第一级明纹”对应 \(k=1\)。

20. 光程差与相位差

光程： \[ \text{光程} = nr \] - 光在介质中传播 \(r\) 距离，等价于在真空中传播 \(nr\) 的距离

光程差 \(\delta\) 与相位差 \(\Delta\varphi\) 的关系： \[ \Delta\varphi = \frac{2\pi}{\lambda}\delta \] - \(\lambda\) — 真空中波长 - \(\delta\) — 光程差

干涉判据的统一形式（用光程差）： - 加强（明）：\(\delta = \pm k\lambda\) - 减弱（暗）：\(\delta = \pm (2k+1)\frac{\lambda}{2}\)

引入光程的意义： - 把光在不同介质中的传播统一折算为真空中的传播 - 有了光程，干涉公式不用区分介质，统一写为上述形式

⚠️ 注意： 波在介质中波长变为 \(\lambda_n = \lambda/n\)，但干涉计算中使用真空波长 \(\lambda\) 与光程差 \(\delta\) 更方便。

21. 薄膜干涉

光路： 光照射到薄膜上，分别在薄膜上下表面反射，两束反射光发生干涉。

光程差公式（反射光）： \[ \delta = 2ne + \frac{\lambda}{2}\ (\text{或}\ 0) \quad\text{取决于是否附加半波损失} \]

  • \(n\) — 薄膜折射率
  • \(e\) — 薄膜厚度
  • 附加项：\(\frac{\lambda}{2}\) 是半波损失带来的额外光程差

半波损失的判断规则： > 光从光疏介质（折射率小）射向光密介质（折射率大）的反射面时，反射光发生 \(\pi\) 相位突变，相当于额外增加 \(\lambda/2\) 的光程差。

判断方法（设薄膜折射率为 \(n_2\)，上下介质为 \(n_1, n_3\)）：

条件 上表面反射 下表面反射 附加光程差
\(n_1 < n_2 < n_3\) 有半波损失 有半波损失 \(0\)（两损失抵消）
\(n_1 < n_2 > n_3\) 有半波损失 无半波损失 \(\lambda/2\)
\(n_1 > n_2 < n_3\) 无半波损失 有半波损失 \(\lambda/2\)

⚠️ 重点关注： 半波损失的条件判断是薄膜干涉的最大易错点！考试中一定会出现。

透射光的干涉： 透射光与反射光的干涉图样互补（反射光减弱对应透射光增强）。

22. 迈克尔逊干涉仪

原理： 利用分振幅法产生干涉。光源发出的光经分光镜分为两束，分别被两个反射镜反射后相遇产生干涉。

等臂时： 两束光光程差 \(\delta = 0\) → 中央亮纹（若不计半波损失）。

移动反射镜 \(M_1\) 移动距离 \(\Delta d\)： \[ \Delta d = N \cdot \frac{\lambda}{2} \] - \(N\) — 干涉条纹移动的数目 - \(\lambda\) — 光波波长

应用——测量微小位移： \[ \Delta d = \frac{\lambda}{2} \cdot \Delta N \] - 可以测量极微小的位移（精度达到波长量级） - 也可以反过来精确测量波长

考试题型： 给出条纹移动数目和波长求位移，或给出位移和条纹移动数求波长。

23. 劈尖干涉

装置： 两块平板玻璃一端接触，另一端夹一根细丝（或薄片），形成空气劈尖。

光程差（反射光）： \[ \delta = 2ne + \frac{\lambda}{2} \] - 空气劈尖中 \(n \approx 1\)，\(\delta = 2e + \lambda/2\)

明暗纹条件： - 明纹：\(2e + \lambda/2 = k\lambda\) → \(e = (2k-1)\lambda/4\) - 暗纹：\(2e + \lambda/2 = (2k+1)\lambda/2\) → \(e = k\lambda/2\)（棱边处 \(e=0\) 为暗纹—因为有半波损失）

相邻条纹对应的厚度差： \[ \Delta e = \frac{\lambda}{2} \]

相邻条纹间距： \[ l = \frac{\lambda}{2\theta} \quad (\theta \text{为劈尖角}) \]

判断凹凸： - 若某处条纹向棱边方向弯曲 → 该处凹下去 - 若某处条纹向背离棱边的方向弯曲 → 该处凸出来

原理：条纹是等厚线，弯曲表示该处厚度与两侧不同。

测量细丝直径/薄片厚度： \[ d = \frac{\lambda}{2} \cdot \frac{L}{l} = \frac{\lambda}{2} \cdot N \] - \(L\) — 劈尖长度 - \(l\) — 条纹间距 - \(N\) — 条纹总数

⚠️ 重点： 半波损失导致棱边为暗纹（\(e=0\) 处），这是检查劈尖干涉是否正确的重要判据。

24. 单缝衍射与光栅衍射

单缝夫琅禾费衍射（半波带法）：

暗纹条件： \[ a\sin\theta = \pm k\lambda,\quad k = 1,2,3,\ldots \]

明纹条件（近似）： \[ a\sin\theta = \pm (2k+1)\frac{\lambda}{2},\quad k = 1,2,\ldots \]

  • \(a\) — 缝宽
  • 中央明纹宽度是其他明纹的两倍
  • 中央明纹半角宽：\(\theta = \lambda/a\)（\(a \gg \lambda\) 时衍射不明显）

光栅衍射：

光栅方程（主极大条件）： \[ d\sin\theta = \pm k\lambda,\quad k = 0,1,2,\ldots \] - \(d = a + b\) — 光栅常数（缝宽+缝间距）

缺级现象： 当光栅方程的 \(k\) 级与单缝衍射暗纹条件同时满足时，该级主极大缺级： \[ \frac{d}{a} = \frac{k}{k'} \] 例如 \(d/a = 3\) 时，\(k = \pm 3, \pm 6, \ldots\) 级缺级。

可观察到的最高级次： \[ k_{\max} = \left\lfloor\frac{d}{\lambda}\right\rfloor \quad (\sin\theta \leq 1) \]

对比： 单缝衍射是干涉和衍射的综合（缝本身的衍射），光栅衍射是多缝干涉受到单缝衍射调制。

⚠️ 易错点： 缺级的计算需要同时考虑光栅方程和单缝衍射暗纹条件，找到公共解。

25. 光的偏振定律

偏振态： - 自然光： 各个方向振动强度相等 - 线偏振光： 只有一个方向振动 - 部分偏振光： 某个方向振动较强

马吕斯定律（线偏振光通过偏振片）： \[ I = I_0 \cos^2 \alpha \] - \(I_0\) — 入射偏振光强度 - \(\alpha\) — 偏振片偏振化方向与入射光振动方向的夹角 - \(\alpha = 0^\circ\)：透射最强；\(\alpha = 90^\circ\)：完全消光

自然光通过偏振片： \(I = I_0/2\)

26. 布儒斯特定律

内容： 当入射角等于某特定值 \(i_B\) 时，反射光成为完全偏振光（振动方向垂直于入射面）。

布儒斯特角公式： \[ \tan i_B = \frac{n_2}{n_1} \] - \(n_1\) — 入射介质折射率 - \(n_2\) — 折射介质折射率 - \(i_B + r = 90^\circ\)（反射光与折射光垂直）

此时： - 反射光：完全偏振光（振动方向垂直于入射面） - 折射光：部分偏振光（以平行于入射面的振动为主）

应用： 激光器输出窗、拍照用偏振滤光片（消除反光）。

对比马吕斯定律： 马吕斯定律描述偏振光通过偏振片后的强度变化；布儒斯特定律描述如何通过反射获得偏振光。

七、气体动理论

27. 理想气体状态方程

常用形式： \[ pV = \frac{m}{M}RT = \nu RT \] \[ p = nkT \]

  • \(p\) — 压强，\(V\) — 体积，\(T\) — 热力学温度（K）
  • \(\nu = m/M\) — 摩尔数
  • \(R = 8.31\ \text{J/(mol·K)}\) — 普适气体常量
  • \(k = R/N_A = 1.38 \times 10^{-23}\ \text{J/K}\) — 玻尔兹曼常数
  • \(n = N/V\) — 分子数密度

变式应用： - 等温：\(p_1V_1 = p_2V_2\) - 等压：\(V_1/T_1 = V_2/T_2\) - 等容：\(p_1/T_1 = p_2/T_2\)

✅ 考点： 给出 \(p\!-\!V\) 图，判断温度高低、气体状态变化。注意 \(T\) 必须是热力学温标（开尔文），不是摄氏温度。

28. 理想气体的压强公式与温度公式

压强公式（宏观微观联系）： \[ p = \frac{1}{3}nm\overline{v^2} = \frac{2}{3}n\overline{\varepsilon_k} \] - \(\overline{v^2}\) — 分子速率平方的平均值 - \(\overline{\varepsilon_k} = \frac{1}{2}m\overline{v^2}\) — 分子的平均平动动能

温度公式： \[ \overline{\varepsilon_k} = \frac{3}{2}kT \] - 温度是分子平均平动动能的量度 - 温度是宏观量，分子热运动剧烈程度的标志

物理意义： - 压强是大量分子对器壁碰撞的统计平均效果 - 温度是分子无规则运动剧烈程度的统计平均量度 - 对单个分子谈温度没有意义

⚠️ 注意： \(p\) 和 \(T\) 都是统计量，对少量分子（如少于几十个）不适用。

29. 能量均分定理

内容： 在热平衡状态下，分子的每个自由度平均分到 \(\frac{1}{2}kT\) 的能量。

分子自由度 \(i\)：

分子类型 自由度 \(i\) 例子
单原子 \(3\)（全平动） He, Ne, Ar
双原子（刚性） \(5\)（3平动+2转动） \(H_2, O_2, N_2\)
多原子（刚性） \(6\)（3平动+3转动） \(CO_2, H_2O\)
双原子（考虑振动） \(7\)（+1振动动能+1振动势能） 高温下

分子平均能量： \[ \overline{\varepsilon} = \frac{i}{2}kT \]

理想气体的内能（\(\nu\) 摩尔）： \[ E = \nu \cdot \frac{i}{2}RT = \frac{m}{M}\cdot\frac{i}{2}RT \]

关键区分： | 对象 | 平均平动动能 | 平均总动能 | 内能（\(\nu\) 摩尔）| |——|————|———–|—————–| | 一个分子 | \(\frac{3}{2}kT\) | \(\frac{i}{2}kT\) | — | | 定量气体（总） | \(\nu \cdot \frac{3}{2}RT\) | \(\nu \cdot \frac{i}{2}RT\) | \(\nu \cdot \frac{i}{2}RT\) |

⚠️ 易错点： - 一个分子的平均平动动能总是 \(\frac{3}{2}kT\)（与分子结构无关！） - 但一个分子的总平均动能 = \(\frac{i}{2}kT\)，与自由度有关 - 理想气体内能 = 分子总动能（无分子间势能）

31. 三种特征速率

注：原 30 空已删除，此为原 31

速率分布函数 \(f(v)\)： \[ f(v) = \frac{dN}{N dv} \] - 物理意义：速率在 \(v\) 附近单位速率间隔内的分子数占总分子数的比例 - 归一化条件：\(\int_0^\infty f(v) dv = 1\)

三种特征速率（从麦克斯韦速率分布律得出）：

速率 公式 用途
最概然速率（最可几速率） \(v_p = \sqrt{\dfrac{2RT}{M}} = \sqrt{\dfrac{2kT}{m}}\) 分布曲线的峰值位置
平均速率 \(\bar{v} = \sqrt{\dfrac{8RT}{\pi M}} = \sqrt{\dfrac{8kT}{\pi m}}\) 分子运动的平均快慢
方均根速率 \(\sqrt{\overline{v^2}} = \sqrt{\dfrac{3RT}{M}} = \sqrt{\dfrac{3kT}{m}}\) 计算平均平动动能

大小关系： \(v_p < \bar{v} < \sqrt{\overline{v^2}}\)

比例关系： \(v_p : \bar{v} : \sqrt{\overline{v^2}} = \sqrt{2} : \sqrt{8/\pi} : \sqrt{3} \approx 1 : 1.13 : 1.22\)

应用： - 计算分子的平均平动动能用方均根速率 - 研究碰撞频率、平均自由程用平均速率 - 分布曲线的峰值对应最概然速率

八、热力学

32. 四种热力学过程的综合计算

热力学第一定律（核心工具）： \[ Q = \Delta E + A \] - \(Q\) — 吸热（\(>0\) 吸热，\(<0\) 放热） - \(\Delta E\) — 内能增量（\(\Delta E = \nu C_V\Delta T\)，对任何过程都成立） - \(A\) — 对外做功（\(>0\) 对外做功，\(<0\) 外界对气体做功）

四个等值过程对比：

过程 特征 内能增量 \(\Delta E\) 做功 \(A\) 热量 \(Q\)
等容 \(V = \text{常数}\) \(\nu C_V\Delta T\) \(0\) \(\nu C_V\Delta T\)
等压 \(p = \text{常数}\) \(\nu C_V\Delta T\) \(p\Delta V = \nu R\Delta T\) \(\nu C_p\Delta T\)
等温 \(T = \text{常数}\) \(0\) \(\nu RT\ln\dfrac{V_2}{V_1}\) \(A\)（全部转化为功）
绝热 \(Q = 0\) \(\nu C_V\Delta T\) \(-\Delta E\) \(0\)

摩尔热容： - \(C_V = \dfrac{i}{2}R\) - \(C_p = C_V + R = \dfrac{i+2}{2}R\) - \(\gamma = \dfrac{C_p}{C_V} = \dfrac{i+2}{i}\)（比热容比）

分子类型 \(i\) \(C_V\) \(C_p\) \(\gamma\)
单原子 3 \(3R/2\) \(5R/2\) \(5/3 \approx 1.67\)
双原子刚性 5 \(5R/2\) \(7R/2\) \(7/5 = 1.40\)
多原子刚性 6 \(3R\) \(4R\) \(4/3 \approx 1.33\)

泊松公式（绝热过程）： \[ pV^\gamma = \text{常量}, \quad V^{\gamma-1}T = \text{常量}, \quad p^{\gamma-1}T^{-\gamma} = \text{常量} \]

吸放热判断（\(p\!-\!V\) 图上）： - 在 \(p\!-\!V\) 图上画绝热线作为参考 - 从同一状态出发，比绝热线”陡”的是吸热方向，比绝热线”平”的是放热方向 - 或直接计算 \(Q = \Delta E + A\) 的正负

⚠️ 易错点： \(\Delta E = \nu C_V\Delta T\) 对任何过程都成立（不限于等容过程），因为理想气体内能只与温度有关。

33. 循环效率

正循环（热机）： 系统从高温热源吸热 \(Q_1\)，对外做功 \(A\)，向低温热源放热 \(Q_2\)。

效率公式（两个形式）：

形式一： \[ \eta = \frac{A}{Q_1} = 1 - \frac{Q_2}{Q_1} \] - \(A\) — 净功（循环曲线包围的面积） - \(Q_1\) — 吸热总和 - \(Q_2\) — 放热总和（取绝对值）

使用场景： 给出了具体的吸热和放热量时使用。

形式二： \[ \eta = 1 - \frac{Q_2}{Q_1} \] 使用场景： 结合热力学第一定律计算 \(Q_1\) 和 \(Q_2\) 时使用。

逆循环（制冷机）： 外界做功 \(A\)，系统从低温热源吸热 \(Q_2\)，向高温热源放热 \(Q_1\)。

制冷系数： \[ e = \frac{Q_2}{A} = \frac{Q_2}{Q_1 - Q_2} \]

⚠️ 注意： \(Q_1\) 是吸热总和（不是净吸热），\(Q_2\) 是放热总和（取绝对值）。要先识别循环中各段是吸热还是放热。

34. 卡诺循环

组成： 两个等温过程 + 两个绝热过程

卡诺热机效率（只与热源温度有关）： \[ \eta_{\text{卡}} = 1 - \frac{T_2}{T_1} \] - \(T_1\) — 高温热源温度 - \(T_2\) — 低温热源温度

卡诺制冷系数： \[ e_{\text{卡}} = \frac{T_2}{T_1 - T_2} \]

卡诺定理： - 所有工作在相同高温热源和低温热源之间的热机，可逆机的效率最高 - 可逆机的效率都等于卡诺热机的效率 - 不可逆机的效率小于卡诺热机的效率

重要推论： \[ \frac{Q_1}{T_1} + \frac{Q_2}{T_2} = 0 \quad (\text{可逆循环}) \] 引入熵变概念： \[ \Delta S = \int \frac{dQ}{T} \quad (\text{可逆过程}) \]

⚠️ 易错点： - 卡诺效率只与热力学温度有关，与工作物质无关 - \(T_1 \to \infty\) 或 \(T_2 \to 0\) 才能使 \(\eta = 1\)，但都不可实现 - 实际热机效率总是小于对应的卡诺效率

九、振动与波

35. 简谐振动的运动学与动力学方程

动力学方程（回复力）： \[ F = -kx \] \[ m\frac{d^2x}{dt^2} = -kx \quad\Rightarrow\quad \frac{d^2x}{dt^2} + \omega^2 x = 0 \]

运动学方程（振动表达式）： \[ x = A\cos(\omega t + \varphi) \]

各量含义： - \(A\) — 振幅（最大位移） - \(\omega = \sqrt{k/m}\) — 角频率（\(\omega = 2\pi/T = 2\pi f\)） - \(\varphi\) — 初相位（\(t=0\) 时的相位） - \(\omega t + \varphi\) — 相位

速度和加速度： \[ v = -\omega A\sin(\omega t + \varphi) = \omega A\cos\left(\omega t + \varphi + \frac{\pi}{2}\right) \] \[ a = -\omega^2 A\cos(\omega t + \varphi) = \omega^2 A\cos(\omega t + \varphi + \pi) \]

能量： \[ E_k = \frac{1}{2}mv^2 = \frac{1}{2}m\omega^2 A^2\sin^2(\omega t + \varphi) \] \[ E_p = \frac{1}{2}kx^2 = \frac{1}{2}kA^2\cos^2(\omega t + \varphi) \] \[ E = E_k + E_p = \frac{1}{2}kA^2 = \frac{1}{2}m\omega^2 A^2 \quad (\text{守恒}) \]

注意： 简谐振动中动能和势能此消彼长，但总机械能守恒。

36. 旋转矢量法

方法： 用一个以角速度 \(\omega\) 匀速旋转的矢量 \(\vec{A}\) 在 \(x\) 轴上的投影来表示简谐振动。

对应关系： - 矢量长度 → 振幅 \(A\) - 旋转角速度 → 角频率 \(\omega\) - 矢量与 \(x\) 轴夹角 → 相位 \(\omega t + \varphi\) - \(t=0\) 时的夹角 → 初相 \(\varphi\)

用途：

(a) 求初相 \(\varphi\)： 已知 \(t=0\) 时的 \(x_0\) 和 \(v_0\) 方向，在旋转矢量图上确定矢量位置。

(b) 比较相位差： - \(\Delta\varphi = 0\)：同相（同时达到最大、同时过零） - \(\Delta\varphi = \pi\)：反相（一个最大时另一个最小） - \(\Delta\varphi = \pm\pi/2\)：相位差 \(\pi/2\)

(c) 求时间间隔： 旋转矢量转过角度 \(\Delta\theta = \omega\Delta t\) → \(\Delta t = \Delta\theta/\omega\)

旋转矢量法是解决简谐振动问题的强大工具，特别是求初相和相位差，比纯代数方法直观得多。

37. 简谐振动的合成

(a) 同方向、同频率的两个简谐振动的合成： \[ x_1 = A_1\cos(\omega t + \varphi_1),\quad x_2 = A_2\cos(\omega t + \varphi_2) \]

合振动仍是同频率的简谐振动： \[ x = A\cos(\omega t + \varphi) \]

合振幅： \[ A = \sqrt{A_1^2 + A_2^2 + 2A_1A_2\cos(\varphi_2 - \varphi_1)} \]

合初相： \[ \tan\varphi = \frac{A_1\sin\varphi_1 + A_2\sin\varphi_2}{A_1\cos\varphi_1 + A_2\cos\varphi_2} \]

特殊情况： - \(\Delta\varphi = 0\)（同相）：\(A = A_1 + A_2\)（干涉加强） - \(\Delta\varphi = \pi\)（反相）：\(A = |A_1 - A_2|\)（干涉减弱） - \(A_1 = A_2\) 且反相：\(A = 0\)（完全抵消）

(b) 同方向、不同频率——拍： 当 \(\omega_1 \approx \omega_2\) 时产生拍现象： \[ x = 2A\cos\left(\frac{\omega_2 - \omega_1}{2}t\right) \cdot \cos\left(\frac{\omega_1 + \omega_2}{2}t\right) \] 拍频：\(v = |\nu_1 - \nu_2|\)

(c) 互相垂直的简谐振动的合成： - 同频率：合成轨迹为椭圆（李萨如图形） - 频率比为简单整数比：形成稳定的李萨如图形

考试重点： 同方向同频率合成是基础，掌握旋转矢量法求合振幅和初相。

38. 波函数

波函数的一般形式： \[ y(x,t) = A\cos\left(\omega t - kx + \varphi\right) \] \[ y(x,t) = A\cos 2\pi\left(\frac{t}{T} - \frac{x}{\lambda} + \varphi\right) \]

  • \(\omega = 2\pi/T\) — 角频率
  • \(k = 2\pi/\lambda\) — 波数（不是玻尔兹曼常数）
  • \(u = \lambda/T = \nu\lambda\) — 波速

各量关系： \[ u = \frac{\omega}{k} = \lambda\nu \]

波函数的物理意义： - 固定 \(x\)：\(y(t)\) 是该点的振动方程 - 固定 \(t\)：\(y(x)\) 是该时刻的波形 - \(x\) 和 \(t\) 都变：描述波的传播

波速（机械波在弹性介质中）： - 绳中横波：\(u = \sqrt{F/\mu}\)（\(F\) 张力，\(\mu\) 线密度） - 固体中纵波：\(u = \sqrt{Y/\rho}\)（\(Y\) 杨氏模量）

波动方程： \[ \frac{\partial^2 y}{\partial x^2} = \frac{1}{u^2}\frac{\partial^2 y}{\partial t^2} \]

⚠️ 注意区分： - \(y\) 表示振动位移，\(A\) 表示振幅 - 波函数中 \(x\) 前的负号表示波沿 \(x\) 正方向传播；正号表示沿 \(x\) 负方向传播

39. 波的干涉与驻波

波的干涉条件： 频率相同、振动方向相同、相位差恒定（相干波）。

干涉强度分布： \[ I = I_1 + I_2 + 2\sqrt{I_1I_2}\cos\Delta\varphi \]

干涉加强/减弱条件： - 加强：\(\Delta\varphi = \pm 2k\pi\) 或 \(\delta = \pm k\lambda\) - 减弱：\(\Delta\varphi = \pm (2k+1)\pi\) 或 \(\delta = \pm (2k+1)\lambda/2\)

波程差与相位差关系： \[ \Delta\varphi = \frac{2\pi}{\lambda}\delta \]

驻波（两列振幅相同、传播方向相反的相干波叠加）：

驻波方程： \[ y = 2A\cos\left(\frac{2\pi x}{\lambda}\right)\cos(\omega t) \]

  • 波腹（振幅最大处）：\(x = \pm k \cdot \lambda/2,\ k = 0,1,2,\ldots\)
  • 波节（振幅为零处）：\(x = \pm (2k+1) \cdot \lambda/4,\ k = 0,1,2,\ldots\)
  • 相邻波腹（或相邻波节）间距：\(\lambda/2\)
  • 相邻波腹与波节间距：\(\lambda/4\)

半波损失在驻波中的体现： 当反射点处出现波节 → 反射时有半波损失 当反射点处出现波腹 → 反射时无半波损失

驻波的特点： - 能量不传播（只在波腹和波节间来回转换） - 各点振幅不同，相位不是传播关系 - 相邻段振动相位相反

⚠️ 易错点： 驻波不是”波”——它不传播能量，各点以相同频率振动但振幅不同。驻波方程是两个反向传播波的叠加结果。
